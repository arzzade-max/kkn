# Kokan NGO — Homepage UI

## Overview
This is the Phase 1 Homepage UI for the Kokan NGO website (kokanngo.org), built as per the Task Assignment document.

## Project Structure
```
kokanngo/
├── index.html          # Main homepage
├── css/
│   └── style.css       # Custom styles
├── js/
│   └── main.js         # JavaScript functionality
├── images/             # Add your images here
└── README.md
```

## Sections Included
- ✅ Hero Banner with campaign CTA
- ✅ Focus Areas (6 Program Cards)
- ✅ Saksham Program Highlight Section
- ✅ Our Footprints (Map + Impact Statistics)
- ✅ Sustainable Development Goals (SDG icons)
- ✅ Impact Statistics (Animated Counters)
- ✅ Stories/Testimonials (Auto Slider)
- ✅ Latest Events (Cards)
- ✅ Moments (Photo Gallery with Filter)
- ✅ Footer (Links, Social, Newsletter, Donation CTA)

## Tech Stack
- **Frontend:** Bootstrap 5.3 + Custom CSS
- **Fonts:** Google Fonts (Playfair Display + DM Sans)
- **Icons:** Font Awesome 6
- **Animations:** AOS (Animate On Scroll)
- **Backend Ready:** Designed for Laravel integration

## Features
- Fully mobile-responsive design
- SEO-friendly semantic HTML
- Smooth scroll & animated counters
- Testimonial slider with touch support
- Gallery filter by category
- Fixed navbar with scroll behavior
- Back-to-top button
- Donation amount selector

## How to Use
1. Open `index.html` in a browser (no build step required)
2. Replace placeholder images in `/images/` with real photos
3. Integrate with Laravel backend for dynamic content
4. Connect Razorpay/Stripe/PayPal for the donation CTA
